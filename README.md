# GraysonRicketts.github.io

Personal portfolio site built with [Astro](https://astro.build) and Tailwind CSS. Live at [graysonricketts.github.io](https://graysonricketts.github.io).

## Development

```bash
npm install
npm run dev      # localhost:4321
npm run build    # production build → dist/
npm run preview  # preview production build
```

## Stack

- **Astro 6** — static site generation
- **Tailwind CSS v4** — styling, zinc/orange palette, dark mode via `.dark` class
- **MDX** — for writing/article pages
- **GitHub Actions** — builds and deploys to GitHub Pages on push to `main`
